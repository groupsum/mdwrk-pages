# Code of Conduct

## Our Pledge
We pledge to make participation in this project a harassment-free experience for everyone.

## Our Standards
Examples of behavior that contributes to a positive environment include:
- Being respectful and constructive.
- Accepting feedback gracefully.
- Prioritizing repository health and documentation integrity.

Examples of unacceptable behavior include:
- Harassment or discrimination.
- Trolling, insulting, or derogatory comments.
- Publishing private information without consent.

## Enforcement Responsibilities
Project maintainers are responsible for clarifying and enforcing standards of acceptable behavior.

## Scope
This Code of Conduct applies within all project spaces and public representations of the project.

## Reporting
Report violations to the maintainers through established repository communication channels.
