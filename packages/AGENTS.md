# MdWrk Pages Package Agents

This folder owns reusable MdWrk Pages packages.

- Keep portable content contracts, compilation, SEO/discovery, themes, React rendering, markdown adapters, page templates, presets, and structured data in package-owned folders.
- Do not place mdwrk.com host behavior or site-specific content here.
- Package changes must preserve public package metadata, `dist` exports, package tests, and patch or minor version increments.
