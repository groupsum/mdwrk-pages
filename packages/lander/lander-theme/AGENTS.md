# Lander Theme Agent Instructions

This package is styles-focused for the public MdWrk site. It should carry lander visual identity without weakening shared theme contracts.

- Use the `styles_specialist` custom agent for delegated lander theme work.
- Keep public-site styles visually distinctive while retaining accessible contrast, predictable typography, and responsive stability.
- Coordinate any token or class contract change with `packages/contracts/theme-contract` and `packages/shared/ui-tokens`.
- Verify visible changes with screenshots for affected routes and responsive bands.
